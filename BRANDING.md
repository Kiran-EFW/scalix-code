## Color Palette

Inherited from **Scalix World** corporate branding - teal/blue gradient system:

```
Primary Colors:
  Primary Teal:    #0D6B75  (rgb(13, 107, 117))
  Secondary Teal:  #06918B  (rgb(6, 145, 139))
  Accent Green:    #45A78F  (rgb(69, 167, 143))

Supporting Colors:
  Dark Blue:       #08204B  (rgb(8, 32, 75))
  Light Teal:      #A8CDCA  (rgb(168, 205, 202))

Grayscale:
  Light Gray:      #F5F5F5
  Medium Gray:     #A0A0A0
  Dark Gray:       #333333
  Black:           #000000
```

**Color Usage**:
- **Primary Teal** (#0D6B75): Headers, primary actions, focus states
- **Secondary Teal** (#06918B): Borders, secondary elements
- **Accent Green** (#45A78F): Success states, highlights
- **Dark Blue** (#08204B): Text, dark backgrounds
- **Light Teal** (#A8CDCA): Backgrounds, subtle accents

## Typography

### Brand Typeface
- **Logo & Headings**: Sans-serif, bold, geometric
- **Body Text**: Clean sans-serif for maximum readability
- **Code**: Monospace (Courier New, Menlo, Monaco)

### Font Sizes & Weights

**Headlines**:
- H1 (Product Name): 32px, Bold
- H2 (Section): 24px, Bold
- H3 (Subsection): 18px, Bold

**Body**:
- Regular: 14px, Normal weight
- Small: 12px, Normal weight
- Code: 13px, Monospace

## Logo Usage Guidelines

### Do's ✅
- Use the primary ASCII logo in terminal applications
- Use on white or light backgrounds (for digital)
- Maintain clear space around the logo (minimum 20px padding)
- Use full brand name "Scalix CLAW" with logo
- Use appropriate color variants for backgrounds

### Don'ts ❌
- Don't distort or stretch the logo
- Don't change the colors without approval
- Don't remove the CLAW acronym explanation
- Don't use pixelated or low-resolution versions
- Don't combine with competing logos

## Web & Dashboard Branding (Phase 3D)

### Color Application
- Header/Navigation: #08204B (dark blue)
- Buttons (Primary): #0D6B75 (primary teal)
- Buttons (Secondary): #06918B (secondary teal)
- Accents/Links: #45A78F (accent green)
- Success States: #45A78F (green)
- Error States: #E74C3C (red)
- Backgrounds: #F5F5F5 (light) or #FFFFFF (white)

### Component Styling
- Rounded corners: 4-8px radius
- Borders: 1-2px, using secondary teal
- Shadows: Subtle, elevation-based
- Spacing: 8px base unit for consistency
