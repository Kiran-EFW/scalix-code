# Scalix Code Press Kit

## One-Liner

Scalix Code is an open-source, self-hosted AI agent orchestration platform for software engineering.

## Short Description (100 words)

Scalix Code is an open-source platform for building, running, and coordinating AI coding agents. Unlike cloud-only solutions, it runs entirely in your infrastructure -- giving teams full control over their data, models, and execution. The platform features multi-agent orchestration with five coordination patterns, an extensible tool system with safety guardrails, production-grade observability, and a plugin architecture. It ships with nine specialized agents for code analysis, security scanning, test generation, bug fixing, and more. Built with TypeScript, it supports Anthropic, OpenAI, Google, and local LLM providers.

## Key Facts

- **License**: MIT (fully open source)
- **Language**: TypeScript
- **Runtime**: Node.js >= 20
- **Architecture**: Monorepo with modular packages
- **Agents**: 9 built-in specialized agents
- **Tools**: Extensible tool system with built-in AST parsing, security scanning, dependency analysis
- **Orchestration**: 5 coordination patterns (sequential, parallel, tree, reactive, mesh)
- **Observability**: Distributed tracing, Prometheus metrics, structured logging
- **Security**: 34 vulnerability detection patterns, input validation, rate limiting, sandboxing
